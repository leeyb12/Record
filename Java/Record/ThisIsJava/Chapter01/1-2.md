# 1.2 운영체제별 JDK 설치

## JDK 설치 및 자바 개발 환경 구성

<strong>JDK(Java Development Kit, 자바 개발 키트)</strong>는 자바 언어에 관련된 라이브러리와 개발 도구들을 포함하여 자바 프로그램을 개발하고 실행하기 위한 필수 도구이다.

<br>

## 1. JDK와 자바 SE(스탠다드 에디션)의 관계

| 개념 | 설명 | 
|------|------|
| 자바 SE(스탠다드 에디션) | 자바 언어의 <strong>스펙(specification, 명세)</strong>으로 자바 표준판을 의미 |
| JDK | 자바 SE 스펙을 구현한 **자바 개발 키트**로, 자바 프로그램 개발과 실행을 위한 툴킷 |

<br>

## 2. JDK 종류 및 차이점

| 항목 | 오픈 JDK (Open JDK) | 오라클 JDK (Oracle JDK) |
|------|---------------------|-------------------------|
| 라이선스 | GNU 라이선스 (오픈소스) | 오라클 별도 라이선스 |
| 사용료 | 무료 | 개발/학습용 무료, 상업용 유료 |
| 개발 소스 공개 의무 | 없음 | 없음 |
| 성능/안정성 | 점차 향상 중, 거의 차이 없음 | 오픈 JDK 기반, 약간 더 안정적이며 응답성 우수 |
| 상업용 비용 부담 | 없음 | 있음 |

- **학습 및 개발용**으로는 두 가지 모두 사용 가능하나, **상업용 배포시 비용 문제**로 인해 보통 오픈 JDK 사용 권장
- 현재는 오라클 JDK가 오픈 JDK를 기반으로 만들어져 큰 성능 차이 없음

<br>

## 3. JDK 다운로드 및 설치 시 고려사항

| 구분 | 설명 |
|------|------|
| 다운로드 사이트 | - 오픈 JDK : 2개 사이트에서 다운로드 가능 <br/> - 오라클 JDK : 오라클 공식 사이트 |
| 권장 버전 | **LTS(Long Term Support, 장기 지원 버전)** 버전 사용 권장 |
| LTS의 의미 | 최초 릴리즈 후 버그 픽스 및 안정화 빌드를 장기간 지원 |
| 주요 LTS 버전 | JDK 8, JDK 11, JDK 17(가장 최신 LTS), 향후 JDK 21 예정 |
| 비LTS 버전과의 차이 | 비LTS는 테스트 용도 적합, 실제 프로젝트는 LTS 버전 권장 |

<br>
